ANTIVIRUS Y VIRUS

Sitio web por Ale, Brandon y Nacho
Argentina Hacking Group

EL VIRUS

El virus es un programa o script que intenta apoderarse
del sistema operativo o controlar
los archivos guardados en el mismo

Existen muchos tipos de virus y programas que te
ayudan a crear payloads para hacer
los mismos, por ejemplo metasploit es una herramienta
muy popular entre hackers y tiene muchos exploits al igual que payloads


EL ANTIVIRUS

El antivirus es un programa que analiza el codigo fuente de un
programa o script para detectar codigo malisioso y eliminar el archivo. Existen muchos
antivirus pero el mejor y mas completo es Kaspersky y
para analizar links y archivos se puede usar la pagina
virustotal.com que es un sitio para analizar archivos y
links para prevenirse de ransomwares, phishing, etc.
